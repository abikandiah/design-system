# Abumble Design System

A production-ready component library with multiple themes and full dark mode support. Built with React, Tailwind CSS, and Radix UI primitives.

---

## Themes

| Theme | Description |
|-------|-------------|
| **Steel** | Metallic & industrial — brushed steel surfaces, electric cyan accents in dark mode |
| **Linen** | Warm & minimal — clean linen tones, candlelit warmth in dark mode |

Switch themes using the **Theme** toolbar button above. Toggle dark mode with the **Color Mode** button.

---

## Installation

```bash
npm install @abumble/design-system
```

## Usage

```tsx
// Import base styles
import "@abumble/design-system/styles.css";

// Import your chosen theme
import "@abumble/design-system/themes/steel.css";
// or
import "@abumble/design-system/themes/linen.css";

// Apply dark mode by adding the .dark class to <html>
document.documentElement.classList.add("dark");
```

```tsx
import { Button, Badge, Card } from "@abumble/design-system";

export function App() {
  return (
    <Card>
      <Badge variant="success">New</Badge>
      <Button>Get started</Button>
    </Card>
  );
}
```
